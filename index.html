My recent projects

Smart Meter Privacy

Preserving consumers' privacy by using a smart communication infrastructure, cryptographic methods and secure signal processing.

Smart Heating System

Control central heating system based on the availability of electricity from a solar panel.

The Greenhouse Project

Supervising humidity and temperature in a greenhouse via web interface. Irrigation, ventilation and heating are controlled on an automatic basis.

Satyros II

An autonomous vehicle exploring its environment using multiple sensors. Based on the collected data, the robot is capable to draw a map and avoid a collision with obstacles. A graduate school project.

Mensch Aergere Dich Nicht

A multi-agent implementation of the famous German board game. Each agent has its own game strategy. Agents are written in Java and C#, communicating via the Simple Object Access protocol (SOAP).
